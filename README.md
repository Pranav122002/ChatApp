# ChatApp
A real-time messaging MERN app for both personal and group chats.

## Requirement
Make keys.js in backend folder and put your keys in this format
```
module.exports = {
  MONGOURI:
    <your mongodb uri>,
  JWT_SECRET: <jwt secret>,
};
```

## Installation

Frontend
```
npm install
npm start
```

Backend
```
npm install
node app
```
